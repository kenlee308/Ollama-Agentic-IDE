# Ollama Agentic IDE
**Origin:** Original Work (Created by Antigravity AI)  
**Type:** Python GUI Application  

**Description:**  
An ultra-minimal Python IDE with integrated local AI for agentic coding, using Ollama to enable file operations and refactoring. Kenny Lee's creation draws from his applied physics and entrepreneurial roots to make AI accessible for workforce upskilling, teaching agentic behaviors in a lightweight environment—supporting DOL's focus on foundational AI skills for industries like manufacturing.

**Key Features:**  
- **Integrated Tester:** Audits model capabilities for educational insights.  
- **Agentic Chat:** AI handles file creation/deletion via tools.  
- **RAG Support:** Attaches files for contextual assistance.  
- **No Electron:** Pure `tkinter` for efficiency.  

**How to Use:**  
1. Prerequisites: Python 3.8+, Ollama.  
2. Dependencies: `pip install ollama schedule`.  
3. Run IDE: `Start Ollama IDE.bat` or `python Ollama_Agentic_IDE_v1_1.py`.  
4. Run Tester: `Start Integrated Tester.bat`.  
5. Chat: Request coding help or file ops.  

**Alignment Note:** Ideal for grants emphasizing local AI tools for ethical, hands-on workforce training in agentic systems.
